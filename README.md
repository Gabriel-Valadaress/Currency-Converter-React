# Currency Converter

This project is a currency converter application that allows users to convert amounts between different currencies in real-time. The application fetches live exchange rates and provides an intuitive interface for quick conversions.

## Technologies Used

- **React** - JavaScript library for building user interfaces
- **Fetch API** - For retrieving currency exchange rates from external API
- **useState Hook** - For managing form state and component data

## API

This project uses the [ExchangeRate-API](https://www.exchangerate-api.com/) to fetch current exchange rates.
